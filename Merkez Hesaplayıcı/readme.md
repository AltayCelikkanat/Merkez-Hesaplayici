# Altay Çelikkanat - Profesyonel Merkez Hesaplayıcı

Altay Çelikkanat, fiziksel sistemlerde çubuk veya blok parçalarının toplam uzunluğu (L) ve kesim noktası (x) baz alınarak merkez kaymasını hesaplayan, modern ve görselleştirilmiş bir masaüstü uygulamasıdır. Mühendislik hesaplamalarını kolaylaştırmak ve hızlı sonuçlar elde etmek amacıyla geliştirilmiştir.

!
## 🚀 Temel Özellikler

- **Hızlı Hesaplama:** Toplam uzunluk (L) ve kesim noktası (x) değerlerini girerek anında merkez kaymasını hesaplar.
- **2D Görselleştirme:** Sistemin yapısını canvas üzerinde anlık olarak çizer.
- **3D İnceleme:** `Matplotlib` entegrasyonu ile sistemin 3D modellemesini ayrı bir pencerede sunar.
- **Veri Yönetimi:** Hesaplama geçmişini JSON formatında dışa aktarabilir veya daha önceki verileri içe aktararak inceleyebilirsiniz.
- **Modern Arayüz:** `CustomTkinter` ile karanlık mod (dark mode) tasarımı.
