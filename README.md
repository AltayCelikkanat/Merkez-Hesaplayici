# Merkez-Hesaplayici
fiziksel sistemlerde çubuk veya blok parçalarının toplam uzunluğu (L) ve kesim noktası (x) baz alınarak merkez kaymasını hesaplayan, modern ve görselleştirilmiş bir masaüstü uygulamasıdır. Mühendislik hesaplamalarını kolaylaştırmak ve hızlı sonuçlar elde etmek amacıyla geliştirilmiştir.
